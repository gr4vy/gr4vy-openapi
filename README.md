# Gr4vy OpenAPI 3.0 Specification

![CI](https://github.com/gr4vy/gr4vy-openapi/workflows/CI/badge.svg)

This repository contains the OpenAPI 3.0 Specification
(OAS3) for the Gr4vy APIs.

## Downloads

The `main` branch of this repository only contains the raw source for the
specification. Although this is valid OpenAPI 3.0, for a combined, resolved
version please have a look at the following builds.

[![OpenAPI](https://img.shields.io/static/v1.svg?label=OpenAPI%203&message=Download&color=grey&labelColor=0361D4&style=for-the-badge&logoColor=white)](https://raw.githubusercontent.com/gr4vy/gr4vy-openapi/openapi-3.0/openapi.v1.json)
[![Swagger
(en)](https://img.shields.io/static/v1.svg?label=Swagger%202&message=Backported&color=grey&labelColor=0361D4&style=for-the-badge)](https://github.com/gr4vy/gr4vy-openapi/blob/swagger-2.0/swagger.v1.json)


## Swagger 2.0

A backport of the API specification to Swagger 2.0 is available on the
[Swagger 2.0](https://github.com/gr4vy/gr4vy-openapi/tree/swagger-2.0) branch.
